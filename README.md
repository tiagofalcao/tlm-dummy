tlm-dummy
=====

A module to bind unused TLM2 sockets.

Build
------------
This module require a SystemC installation with pkg-config support.

1. Build
```bash
make
```

2. Build and Test
```bash
make test
```

Installation
------------

```bash
make install PREFIX=/usr/local/
```
